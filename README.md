# 智慧教室Web端

## 依赖

```shell
"dependencies": {
  "axios": "^0.18.0",
  "echarts": "^4.2.1",
  "element-ui": "^2.8.2",
  "fetch": "^1.1.0",
  "highcharts": "^7.1.1",
  "vue": "^2.5.2",
  "vue-highcharts": "^0.1.0",
  "vue-resource": "^1.5.1",
  "vue-router": "^3.0.1"
},
```

## run

```shell
cd ahoj-web/ # 进入项目目录

npm install # 安装依赖

npm run dev / npm start # 开发模式运行
```

## build

```shell
npm run build
```
